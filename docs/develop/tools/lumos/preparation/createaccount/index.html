Create Accounts

In this guide, we will prepare two CKB accounts, Alice and Bob, that will be used in the accounts.ts file in the later Hello Lumos example. For more information about the concepts of CKB accounts, CKB capacity, and the methods to create accounts, see CKB Accounts and Capacity.

Prerequisitesâ€‹

The following prerequisites apply for creating the accounts:

  • The CKB node is installed and running on DEV chain. For more information, see Install a CKB node on DEV Chain by Using Tippy.

Create the Accounts by Using ckb-cliâ€‹

Step 1. Download the CKB pre-built installer package.â€‹

The CKB pre-built installer package includes the ckb-cli tool. For more information, see Download the CKB Pre-built Installer Package.

This guide uses the CKB 0.39.0 version. All releases can be found on the CKB releases page. If the package is already downloaded, go to step 2 directly.

Step 2. Verify the ckb-cli tool is working and check the version.â€‹

Navigate into the folder where the ckb-cli tool locates, then verify the tool is working.

ckb-cli -V
ckb-cli 0.39.0

Step 3. Create the account for Alice.â€‹

Step 4. Get the private key for the account of Alice.â€‹

The extended private key (a private key and a chain code) is exported to the alice file under the current working directory. The first line in the file is the private key of the account. The second line is the chain code.

ckb-cli account export --extended-privkey-path alice --lock-arg 0x7e00660b8ab122bca3ba468c5b6eee71f40b7d8e
Password: 
message: "Success exported account as extended privkey to: \"alice\", please use this file carefully"

Step 5. Get CKB capacity for the account of Alice.â€‹

For more information about CKB capacity, see CKB Capacity of an Account.

To get CKB capacity for Alice on DEV chain, specify Alice as the miner who receives mining rewards.

Assign the lock args of Alice to Block Assembler Lock Arg in the Edit Chain form and save the changes.

After the miner is specified, restart the CKB node and start the CKB miner on the Tippy dashboard.

note

CKB is mature 4 epochs after being mined. In Nervos, an epoch is approximately four hours. For DEV chain, the epoch length is defined by the genesis_epoch_length parameter in the dev.toml file. For more information about shortening the epoch length for DEV chain, see Step 4 of Install a CKB node on DEV chain by using Tippy.

Step 6. Check the balance for the account of Alice.â€‹

To check the balance of Alice, execute the ckb-cli wallet get-capacity --address <the Testnet address of the account> command.

ckb-cli wallet get-capacity --address "ckt1qyq8uqrxpw9tzg4u5waydrzmdmh8raqt0k8qmuetsf"
immature: 8039446.8758295 (CKB)
total: 10451302.54823011 (CKB)
